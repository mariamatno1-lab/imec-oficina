# Imec Oficina Mecânica

Site institucional da Imec Oficina Mecânica, desenvolvido em Next.js e preparado para publicação na Vercel.

## Requisitos

- Node.js 20.9 ou superior
- npm

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Validação de produção

```bash
npm run build
npm start
```

## Publicação na Vercel

1. Envie este projeto para um repositório no GitHub.
2. Acesse `https://vercel.com/new`.
3. Importe o repositório.
4. Confirme o framework **Next.js**.
5. Mantenha os comandos e diretórios detectados automaticamente.
6. Clique em **Deploy**.

As páginas públicas são `/` e `/servicos`.
